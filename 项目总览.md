## 音质与延迟调优已生效（2026-07-09 ~ 2026-07-12）

### 音质优化（07-09）

| 改动 | 位置 | 效果 |
|------|------|------|
| 7kHz LPF 关闭 | Windows `AudioEngine.cs` | 保留高频细节，防啸叫移至 Android 端 HPF |
| SpeakerGain 1.8→1.0 | Windows `AudioRouter.cs` | 防止削波失真 |
| WaveOutEvent 保留 | Windows `AudioRouter.cs` | 热切正常的关键（WasapiOut 异步释放导致卡死） |
| HPF 200Hz 高通 | Android `MicrophoneCapturer.kt` | 去风扇/机箱低频底噪，替代 LPF |
| Opus bitrate 64k→128k | Android `AudioEncoder.kt` | 音质提升 |
| complexity 5→10 | Android `AudioEncoder.kt` | 编码质量提升 |
| FEC 开启 | Android `AudioEncoder.kt` | 前向纠错 + CVBR 留余量 |

### 延迟调优（07-12）

| 参数 | 改前 | 改后 | 收益 |
|------|------|------|------|
| Win BufferMs | 300ms | **100ms** | -200ms |
| WaveOutEvent DesiredLatency | 80ms | **100ms** | 稳定值（50 卡顿） |
| WasapiOut CABLE Input | 80ms | **50ms** | -30ms |
| Android 缓冲区 | 未动 | 未动 | 不影响稳态延迟 |
| **端到端估算** | **~400ms** | **~200ms** | ✅ |
